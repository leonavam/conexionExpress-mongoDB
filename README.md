# conexionExpress-mongoDB

## backend-server

- Primero hay que instalar los paquetes de node
```
npm install
```
- Para ejecutar el servidor utilizar:
```
npm start
```

## googleSignin-demo
Es un servidor de prueba para probar la autenticación de Google, un front-end básico, si funciona.

- Se ejecuta con el commando
```
npm run dev
```
